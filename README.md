## Getting Started
Implemented and tested on Ubuntu 18.04 with Python 3.6.

1. Clone this repo
```bash
git clone https://github.com/ch-lai/TWStock-Forecast.git
```

2. Install Python dependencies
```bash
cd TWStock-Forecast
pip3 install -r requirements.txt
```

3. Run the scripts
```bash
python main.py
```
